Everyday GIT Befehle
git init
git status
git add file
git commit -am "initial commit" 
(autotrack changes to staging area, command line message)
EDIT
git commit -am "some changes"
TAGGING
git tag v1
HISTORY
git log --pretty=format:"%h %ad | %s%d [%an]" 
--graph --date=short

[.gitconfig] 
[alias]
  hist = log --pretty=format:\"%h %ad | %s%d [%an]\" --graph --date=short
BRANCH
git branch name
git checkout name (-b creates the branch on the fly)
UNDOING
git reset HEAD 
git revert HEAD
git diff HEAD
(soft doesn't change working dir /
hard changes index and wd)
